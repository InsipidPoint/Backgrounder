This is the type of display shown when Backgrounder is invoked.

Simple Mode

An alert pops-up stating that backgrounding has been activated, or deactivated if backgrounding had previously been enabled.

After a brief pause, the alert will dismiss itself and the application will minimize. If backgrounding has just been activated, the application should now be running in the background. If deactivated, the application should no longer be running.

Task List Mode

A list is presented showing the currently running (GUI-based) applications. The list allows for quickly switching to another running application, as well as to close those applications that are no longer needed (and thus free-up memory).

For information on how to use the task list, please see the task list's built-in help screen.
