THIS IS A BETA VERSION

Many things have changed in this version. Use at your own risk.

Please report any issues with the changes listed below, or with confusing documentation. When reporting, please include your device and firmware version.

Please *do not* report old bugs (bugs that existed in previous versions).

Changes:

  • NEW: Added option to show a backgrounding status indicator in the status bar.
  • NEW: Can now disable backgrounding of apps such as Safari by setting backgrounding method to Off.
  • MOD: Major reworking of preferences
    • All settings can now be set per-application (by creating an "override").
    • "Backgrounding Persists" has been renamed to "Stay Enabled".
    • The former blacklisting option is now acheived by setting "Backgrounding Method" to Native.
    • The former always-enabled option is now acheived by enabling both "Enable at Launch" and "Stay Enabled".
  • MOD: Fixed support for modifying badge via WinterBoard. Modify by overriding image Backgrounder_Badge.png in SpringBoard bundle.
